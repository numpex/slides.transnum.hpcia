            \begin{verbatim}
\section{Elements}
            \end{verbatim}
            The theme provides sensible defaults for:
            \begin{itemize}
              \item \emph{Emphasis} on key text.
              \item \alert{Accents} on important parts.
              \item \textbf{Bold} for results.
              \item Also \textbf{\emph{combine}} the previous ones.
            \end{itemize}
